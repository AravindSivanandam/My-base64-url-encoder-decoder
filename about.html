⁉️ About

  • Base64 encoding defines how to represent binary data in an ASCII string format.

  • The objective of base64 encoding is to transmit binary data such as keys or digital certificates in a printable format.

  • This type of encoding is needed if these objects are transported as part of an email body, a web page, an XML document, or a JSON document.

  • base64url encoding is almost the same, yet different than base64 encoding.

  • Base64url encoding is basically base64 encoding except they use non-reserved URL characters

  • (e.g. – is used instead of + and _ is used instead of /) and they omit the padding characters.

How it Works?

    After the text / message is placed on the Encoder ,This web application converts the message into a non-readable format.

    When the non-readable message is given to the Decoder,this Web application converts it back to a readable message.
